Find your key in the Wattpilot export URL — it's the value of the ?e= query parameter.

🔒 How the key is stored

  • • Saved to config.json on the server — never in your browser or a cookie.
  • • Only the server process can read this file.
  • • The key is only used server-side to fetch data from the Wattpilot API.
  • • The WATTPILOT_KEY environment variable still works as a fallback.
